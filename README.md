# JLPT Study Library

A complete JLPT study resource library covering N5 through N1.

## Contents

- **Beginner Guide** — Hiragana, katakana, learning method, first week plan
- **N5 Integrated Workbooks** (Vol I, II, III) — 901 vocab, 158 kanji, 24 lessons
- **N5–N4** Complete guides, reference volumes, workbooks, short stories
- **N3–N2** Complete guides, reference volumes, workbooks
- **N1** Complete study guide

## Hosting on GitHub Pages

1. Create a new GitHub repository (e.g. `jlpt-library`)
2. Upload all files — keep `index.html` at the root and all PDFs inside the `pdfs/` folder
3. Go to **Settings → Pages → Source → main branch → / (root)**
4. Your library will be live at `https://yourusername.github.io/jlpt-library`

## File Structure

```
index.html          ← the library UI (40KB)
pdfs/
  JLPT_Beginner_Guide.pdf
  JLPT_N5_Integrated_Workbook.pdf
  JLPT_N5_Integrated_Workbook_Vol2.pdf
  JLPT_N5_Integrated_Workbook_Vol3.pdf
  ... (35 PDFs total)
```

## Embedding in Google Sites

Once hosted on GitHub Pages, embed it in Google Sites:
1. In Google Sites, click **Insert → Embed**
2. Paste your GitHub Pages URL
3. The library will load inside your Google Site as a live iframe
